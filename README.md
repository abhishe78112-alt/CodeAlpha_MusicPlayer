# 🎵 CodeAlpha Music Player

A modern and responsive Music Player built using **HTML, CSS, and JavaScript** as part of the **CodeAlpha Frontend Development Internship**.

## 🚀 Features

- Play / Pause Music
- Previous / Next Song
- Playlist
- Song Title & Artist
- Progress Bar
- Current Time & Duration
- Volume Control
- Auto Play Next Song
- Keyboard Shortcuts
- Dark / Light Mode
- Responsive Design
- Glassmorphism UI

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
